# Forecast Financiero de Series de Tiempo

## 1. Problema de Negocio
Predecir precios y volatilidad de activos financieros permite:
- Gestionar riesgo de portafolio  
- Optimizar estrategias de inversión  
- Planificar liquidez

## 2. Dataset
- Fuente: Kaggle, S&P 500 Stock Data  
- Contiene precios diarios, volumen, y datos históricos de todas las acciones y ETFs de EE. UU.

## 3. Variables Clave
- Date: fecha de cada observación  
- Close: precio de cierre  
- Volume: volumen negociado  
- Opcionales: open, high, low, adj close

## 4. Modelos Aplicados
1. ARIMA / SARIMA → baseline clásico  
2. GARCH → modelar volatilidad  
3. LSTM / Temporal CNN → forecast avanzado

## 5. Resultados
- Comparación de modelos clásicos vs deep learning  
- Visualización de predicciones vs valores reales  
- Evaluación de desempeño con MAE y RMSE

## 6. Conclusión Ejecutiva
Permite anticipar movimientos de mercado y gestionar riesgo de portafolio con métodos avanzados de series de tiempo.
